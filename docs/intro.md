# شروع

تمام درخواست ها به

```
https://example.com/v1/APIKEY/METHOD
```

برای دریافت کلید ای‌پی‌آی به پنل کاربری خورد مراجعه کنید

متد ها شامل :

### ارسال تکی

- send

### ارسال چندتایی

- sendArray

### ارسال نظیر به نظیر

- sendPair

### ارسال با ساختار از پیش تعریف شده

- sendPattern
